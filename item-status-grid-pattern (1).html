03 — Rules
Do's and Don'ts
Non-negotiable standards across all 30 products.
✅ Always do this
✓Always show stuck items first — never buried below running items
✓Always show the stuck alert banner at the top of the detail page
✓Always show a Fix button directly on the stuck item
✓Always use the 5 shared agent states — never invent new ones
✓Always tell the user which agent caused the stuck state and why
✓Always sort project list by urgency — stuck projects always first
❌ Never do this
✗Never hide stuck items — always visible without filtering
✗Never use a different color for Stuck — always red, no exceptions
✗Never show only a count — always show which product is stuck
✗Never show Fix button without explaining what needs fixing
✗Never sort alphabetically by default — urgency first always
✗Never collapse the stuck section — always expanded
