Input type reference — when to use each
✏️ Text
Simple direct instructions. Default for all users.
📎 File
When the brief or data is in a document.
📋 Template
Repeating run types. Reduces effort and errors.
🕐 History
Reuse or modify a previous instruction.
⚙️ Form
Complex instructions with many parameters.
🎙️ Voice
Long instructions, mobile, or hands-free.
🔗 URL
When the brief lives in a document or web page.
✦ Meta
Sophisticated prompting via goal, blocks, or context assembly.
