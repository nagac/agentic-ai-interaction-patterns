03 — Agent State Indicators
The visual language — used consistently everywhere
Five states. One visual language. Used in every variant, every screen, every product. This is Pattern 3 — Agent State Indicators — embedded inside this pattern.
⏳
Running
Agent is actively working right now
✓
Complete
Agent finished successfully
⚠
Stuck
Agent hit a problem — needs you
⏸
Waiting
Paused — needs your approval
—
Not started
Hasn't reached this stage yet
⚠ Critical rule
These five states and their exact visual treatments must be used consistently across every screen in every product. Never invent a sixth state without updating this pattern first.
04 — Rules
Do's and Don'ts
Non-negotiable standards every designer must follow.
✅ Always do this
✓Always show stuck items first — above running, above complete
✓Use red prominently for stuck — users must not miss it
✓Show a filter to isolate stuck items — especially for large projects
✓Give a direct action button next to stuck items — "Fix now" not just information
✓Use the same five state indicators across all variants and screens
✓Show counts in X/Y format — "8/10" not "80% complete"
❌ Never do this
✗Hide stuck items below running or complete items
✗Use subtle or muted colour for stuck — users will miss it
✗Show only a count of issues without saying which items are affected
✗Invent new state indicators without updating the shared language
✗Show percentage completion only — always show the raw count too
✗Auto-collapse stuck items — they must always be visible
05 — Metrics
How you know this pattern is working
Measure these after the pattern ships.
Time to find stuck item
<10s
How quickly user identifies which product is stuck after opening the project.
🚨 Red flag if over 30s
Layout isn't surfacing issues clearly enough
Filter usage rate
>40%
Users who use the stuck filter on projects with 20+ items.
🚨 Red flag if under 10%
Users not aware of filter — add onboarding nudge
Return visit rate
<3x
How many times user opens the project page per task. Lower is better — means they found what they needed fast.
🚨 Red flag if over 6x
Users are confused — information not clear enough
